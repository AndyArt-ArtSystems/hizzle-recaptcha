== Description ==

**This is a lightweight Google reCAPTCHA plugin for WordPress and WooCommerce.**
★★★★★<br>

It allows you to protect your:-

- WordPress / WooCommerce login forms.
- WordPress / WooCommerce / BuddyPress registration forms.
- WordPress / WooCommerce lost password forms.
- WordPress / WooCommerce reset password forms.
- WordPress comment forms.
- WooCommerce checkout forms.
- Noptin newsletter subscription form.
- BBPress new topic and reply forms.
- BuddyPress new group forms.
- Contact Form 7 forms.
- Mailchimp 4 WordPress forms.
- WPForms forms.
- WPForo new topic and reply forms.

reCAPTCHA can be individually enabled / disabled on each form.

== Installation ==

* Go to your WordPress Dashboard.
* Click on Plugins -> Add New
* Search form "**Hizzle reCAPTCHA**"
* Find the plugin and click on the Install Now button
* After installation, click on Activate Plugin link to activate the plugin.

== Frequently Asked Questions ==
= Will this plugin work with version 3 keys? =

No. We only support version 2 keys for the time being.

Yes, as long as your theme follows the WordPress theme development guidelines.
